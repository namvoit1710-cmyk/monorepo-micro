## Quick Start

```bash
# 1. Install dependencies
bun install

# 2. Start all apps
bun run dev

# 3. Open shell in browser
open http://localhost:3000
```

---

## Key Technologies at a Glance

- **React 19** + **TypeScript 5.9**
- **Module Federation v2** (micro frontend)
- **Rsbuild** (Rspack-based bundler)
- **Turborepo** (monorepo task orchestration)
- **TanStack Query v5** + **tRPC v11**
- **Tailwind CSS v4** + **shadcn/ui**
- **Zod v4** + **React Hook Form v7**
- **Rete.js v2** (workflow editor)
